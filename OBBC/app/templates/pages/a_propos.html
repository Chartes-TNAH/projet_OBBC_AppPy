Le projet

 Cette application web à été développée dans le cadre du cours de développement applicatif du master "Technologies numériques appliquées à l'histoire" de l'École nationale des chartes.

 La consigne initiale était de développer une application utilisant des fichiers XML-TEI pour présenter le corpus.

 Les données sont majoritairement issues de Wikimedia. Il a été décidé, pour les besoins de l'exercice, de ne choisir que dix chansons parmi les quatre thèmes qui composent l'édition de 1846 du Barzaz Breiz.

 L'application web s'appuie sur Python 3.x, le micro-framework Flask et les langages du web tels que HTML, CSS et Javascript. Le dataset XML est constitué de chansons encodées suivant les guidelines TEI. Le langage XSL est appelé dynamiquement pour transformer le fichier XML selon le contexte de navigation. Concernant l'habillage du site, nombres d'éléments proviennent du framework Bootstrap 4. Une base de données a été constituée à partir de SQLlite et est géré par Python grâce à l'ORM SQLAlchemy qui permet de faire le lien avec le dataset XML et utile pour la fonction de recherche plein-texte. Zotero a permis de générer la bibliographie en HTML.


Installation de l'application

    1. Installer OBBC à partir de la branche master du dépôt Github de l'application : $ git clone https://github.com/Lucaterre/projet_OBBC_AppPy.git

    2. Installer Python via le site. Pour rappel : la plupart des systèmes Linux, intègre déjà Python.

    3. Créer un environnement virtuel à l'aide de VirtualEnv. Dans votre terminal taper la commande : $ pip install virtualenv pour installer VirtualEnv puis $ virtualenv -p python3 env ou sous Windows : $ virtualenv -p puis $ env:python3 env.
    4. Activer l'environnement virtuel via la commande $ source env/bin/activate. Pour quitter l'environnement virtuel taper simplement
      $ deactivate

    5. Dans le terminal, se placer au niveau du fichier requirements.txt, puis installer les différents packages nécéssaires avec
      la commande suivante : $ pip install -r requirements.txt

    6. Dans le terminal rentrer la commande $ cd OBBC/. Une fois dans le dossier lancer l'application avec la commande
      $ python run.py ou $ python3 run.py via le serveur local et selon votre version de python ($ python --version ou -V`)

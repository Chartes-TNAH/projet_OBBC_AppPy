Le projet

 Cette application web à été développée dans le cadre du cours de développement applicatif du master "Technologies numériques appliquées à l'histoire" de l'École nationale des chartes.

 La consigne initiale était de développer une application utilisant des fichiers XML-TEI pour présenter le corpus.

 Les données sont majoritairement issues de Wikimedia. Il a été décidé, pour les besoins de l'exercice, de ne choisir que dix chansons parmi les quatre thèmes qui composent l'édition de 1846 du Barzaz Breiz.

 L'application web s'appuie sur Python 3.x, le micro-framework Flask et les langages du web tels que HTML, CSS et Javascript. Le dataset XML est constitué de chansons encodées suivant les guidelines TEI. Le langage XSL est appelé dynamiquement pour transformer le fichier XML selon le contexte de navigation. Concernant l'habillage du site, nombres d'éléments proviennent du framework Bootstrap 4. Une base de données a été constituée à partir de SQLlite et est géré par Python grâce à l'ORM SQLAlchemy notamment pour la fonction de recherche plein-texte. Zotero a permis de générer la bibliographie en HTML.


Installation de l'application

    1. Installer OBBC à partir du dépôt Github de l'application

    2. Installer Python via le site. Pour rappel : sur la plupart des systèmes Linux, Python est pré-installé

    3. Installer un environnement virtuel à l'aide de VirutalEnv. Dans votre terminal taper la commande$ pip install virtualenv pour installer VirtualEnv puis $ virtualenv -p python3 env ou $virtualenv -p puis $env:python3 env(sous Windows). Enfin pour activer l'environnement virtuel, on utilise la commande $ source env/bin/activate. Pour quitter l'environnement virtuel on utilise la commande
      $ deactivate

    4. Dans le terminal, se placer au niveau du fichier "requirements.txt", puis installer les différents packages nécéssaires avec
      la commande suivante : $ pip install -r requirements.txt

    5. Dans le terminal rentrer la commande $ cd OBBC/. Une fois dans le dossier lancer l'application avec la commande
      $ python run.py ou $ python3 run.py via le serveur local
